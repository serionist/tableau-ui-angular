Checkbox

The checkbox component is a simple component that allows users to select or deselect an option.
Supports both Reactive Forms and simple value binding.

Module

To use the component, import one of the following modules into your module/component:

  • TableauUiCheckboxModule - for granular import
  • TableauUiAllModule - to import all components from this library

Basic usage

To use the checkbox without Reactive Forms, simply bind to the value and valueChanges properties. These examples show a snack when changed.
