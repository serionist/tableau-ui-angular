Scroller

The scroller component is used to provide arrow scrolling for overflowing content.

Module

To use the component, import one of the following modules into your module/component:

  • TableauUiArrowScrollModule - for granular import
  • TableauUiAllModule - to import all components from this library

Usage

The ArrowScrollComponent exposes the following properties:

  • scrollType - either default or arrow, can be used to turn on the arrow scrolling behavior or stick to the default scrollbar.
    When arrow is selected, the default scrollbar is turned off and Arrow buttons are shown for scrolling
  • scrollDirection - either horizontal or vertical.
    When horizontal is selected, the default mousewheel scrolling scrolls horizontally even if the scrollType is default
  • buttonsGap - the gap between the arrow buttons and the scroll content
  • buttonsColor - the color property of the scrolling arrow button. This is the primary, secondary, error or plain (default)
  • buttonsCssColor - the actual CSS color of the arrow in the scrolling button
  • buttonScrollAmount - how many pixels to scroll when the arrow button is clicked. Default is 100px.
  • showScrollButtonSeparator - set to true to show a separator between the arrow button and the scroll content. Default is false
