Tooltips

Tooltips are used to display additional information when a user hovers over an element.

Module

To use the component, import one of the following modules into your module/component:

  • TableauUiTooltipModule - for granular import
  • TableauUiAllModule - to import all components from this library

Basic information

A tooltip can be attached to any element; icons, buttons, texts, etc
A tooltip can be positioned in 4 ways: top, bottom, left, right
A tooltip can have a margin of 0, 2em, 10px, 5vh, etc
A tooltip can be provided as text or as a template
A tooltip will automatically adjust its position if it is too close to the edge of the screen
