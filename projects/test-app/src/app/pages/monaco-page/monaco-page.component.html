Monaco Editor

This page explains how to use built-in Monaco Editor

Using Monaco Editor

  1. Update your projects angular.json to include the Monaco Editor assets: :
  2. Import the TableauUiMonacoModule in your component/module
  3. Use the tab-monaco-editor component.

Parameters

The tab-monaco-editor component accepts the following parameters:
  • options : Configures monaco. Adds line numbering, scrollbars, etc
  • uri : The URI of the file to be edited. Used to uniquely identify the file
  • language : The language of the file to be edited. Used to set the syntax highlighting
  • value : The value of the file to be edited. This supports two-way binding
  • theme : The theme of Monaco Editor. Keep it on auto to swap between light/dark configuration of ThemeService
  • onMonacoLoad : You can define a function here that gets called when Monaco Editor loads. You can use it to set JsonOptions, etc.
  • disposeModelOnDestroy : true by default. Destroys the document model when the component is destroyed. Set it to false if you want to load the document later from another editor. Models are kept globally by Monaco Editor using their URIs
  • editorLoaded : output function that gets called after the Editor is online
You can also use the monaco-editor with Reactive Forms ( [formControl] or formControlName )
