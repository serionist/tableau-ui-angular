Radio Groups

The radio groups component is a simple component that allows users to select one option from a list of options.
Supports both Reactive Forms and simple value binding.

Module

To use the component, import one of the following modules into your module/component:

  • TableauUiRadioGroupModule - for granular import
  • TableauUiAllModule - to import all components from this library

Basic usage

To use a radio group without Reactive Forms, simply bind to the [value] and (valueChanges) properties.
These examples show a snack when changed.

Reactive Forms

To use the radio group with Reactive Forms, bind to the formControl or formControlName property.
