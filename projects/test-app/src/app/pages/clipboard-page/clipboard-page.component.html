Clipboard

The clipboard component is used to copy text to the clipboard.

Module

To use the component, import one of the following modules into your module/component:

  • TableauUiClipboardModule - for granular import
  • TableauUiAllModule - to import all components from this library

Usage

You can use the ClipboardService to read from and write to the Clipboard.
It automatically asks for browser permission to access the clipboard.
If browser permission is not given, it will either handle the error by showing an error snack or it will let you handle it.

It is recommended to play around browser permissions for this site and see how the below buttons behave
