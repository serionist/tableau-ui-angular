Date Picker

Date Picker component is a custom form control that allows users to select a date/date+time

Module

To use the component, import one of the following modules into your module/component:

  • TableauUiDatePickerModule - for granular import
  • TableauUiAllModule - to import all components from this library

When component is datetime, the displayed value is always in local time. The .toISOString() method always converts it to UTC though
