Importing Fonts

Tableau UI supports the following fonts in this priority order:
  • Benton Sans, a proprietary font. NOTE: You must have a license to use it.
  • Roboto, an open source Google Font
  • Arial, a free font already included in your browser

To use Benton Sans

You must have a license to use it. If you do not have a license, you can use Roboto or Arial.
  1. Place the following Benton Sans fonts into a public (web hosted) directory:
    • BentonSans-Book.woff
    • BentonSans-Book.woff2
    • BentonSans-Book_Italic.woff
    • BentonSans-Book_Italic.woff2
    • BentonSans-Medium.woff
    • BentonSans-Medium.woff2
    • BentonSans-SemiDemi.woff
    • BentonSans-SemiDemi.woff2
    • BentonSans-SemiDemi_Italic.woff
    • BentonSans-SemiDemi_Italic.woff2
    Suggested directory: public/fonts
