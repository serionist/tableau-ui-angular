Icons

This page demonstrates the use of Material Icons in the component library.

Getting started

Material Design Icons is an established icon set that is widely used in web development.
The component library provides a wrapper around these icons to make them easier to use.
There are two major ways that material icons can be added into your project:

  • Locally Hosted icons , imported via the Material Icons NPM Package:
    All the icons are hosted locally
    Simple installation and usage
    Large size. Browser downloads at least ~4.5MB+ (uncompressed) to make icons available. May be a lot more if multiple icon styles are used (outlined/rounded/sharp)
  • Dynamic Icons . Only the used icons are downloaded at build-time:
    Only loads the used icons, small download size (a few kilobytes, depending on the number of icons used)
    Harder to set up and use. Each used icon needs to be recorded by the developer.

This sample app uses Dynamic Icons, but contains commented code to use Locally Hosted icons as well.

Usage

To add an icon to your app, use the <tab-icon> component and set the value attribute to the name of the icon you want to use.

This will render:
  • You can search for icons on Material Design Icons . Use the icon name as it appears in the Inserting the icon section on the right side panel after an icon is selected.
  • The size of the icon is 1em by default. Changing the font-size CSS property resizes the icon. No need to update width and height .
  • Icons can be colored using the color attribute. Possible values: primary , error , success . For any custom color, the color CSS property can be used.
  • Add type="Material Symbols Outlined" to change the type of the fonts. Avaialble types:
    • Material Symbols Rounded - default
    • Material Symbols Outlined
    • Material Symbols Sharp
  • Add [fill]="true" to make the icon filled.
  • Add [weight]="100" to change the weight of the icon. The default weight is 400, possible values are: 100, 200, 300, 400, 500, 600, 700.
  • Add [grade]="200" to change the grade of the icon. The default grade is 0, possible values are: -25, 0, 200.
  • Add [opticalSizePx]="40" to change the optical size of the icon. The default optical size is 24, possible values are: 20, 24, 40, 48.
